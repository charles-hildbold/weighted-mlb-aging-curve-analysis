# weighted-mlb-aging-curve-analysis
Quantifying MLB Batting Decline using R (2000-2024). This project implements a Weighted Delta Regression model to quantify the average year-over-year decline in $\mathbf{wOBA}$ (Weighted On-Base Average), providing a stable aging factor for player projection systems. Built using R (Lahman, tidyverse) with results visualized in Tableau.
